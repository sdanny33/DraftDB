** Description **
Stat tracking database for Pokemon Draft. The database contains matches from my own draft league, Smogon tournaments and saved replays from the Pokemon Showdown server 
dating back to the start of the year.

** How to use **
The replay scraper scrapes the saved draft replays from the Pokemon Showdown server. The create DB is what intially creates the DB and updates the stats using methods 
outlined in ther parser.
