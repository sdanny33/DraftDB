# DraftDB

## Author

Daniel Soares. Email me if you have questions ([dcs3personal@gmail.com](mailto:dcs3personal@gmail.com)).

## Description

Stat tracking database for Pokémon Draft. The database contains matches from my own draft league, Smogon tournaments and saved replays from the Pokémon Showdown server dating back to the start of the year.

## How to use

The replay scraper scrapes the saved draft replays from the Pokémon Showdown server. The create DB is what initially creates the DB and updates the stats using methods outlined in the parser.
